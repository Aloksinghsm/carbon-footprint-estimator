About the Software

The Flexible Pavement Carbon Footprint Estimator is a desktop software that calculates layer-wise and total carbon emissions using AI/ML models. Designed for civil engineers, academicians, and pavement researchers, it simplifies CO₂ estimation for sustainable pavement design.

Features

  • AI/ML-based carbon emission prediction
  • Fully offline usage – no internet required
  • Automatic backend startup
  • Layer-wise & total emission breakdown
  • Professional PDF report generation
  • User-friendly interface
