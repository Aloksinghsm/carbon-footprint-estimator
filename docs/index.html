About the Software

The Flexible Pavement Carbon Footprint Estimator is a desktop application that calculates layer-wise and total carbon emissions using AI/ML models. It is designed for civil engineers, academicians, and pavement researchers to simplify carbon estimation for sustainable pavement design.

⚠️ IMPORTANT – First Time Setup

After installation:

1️⃣ Go to:
C:\Program Files\Flexible Pavement Dashboard\app\

2️⃣ Double-click carbon_backend.exe
Wait until you see:
"Uvicorn running on http://127.0.0.1:8000"

3️⃣ Leave the window open (you can minimize it)

4️⃣ Now launch the dashboard using the desktop shortcut

✔ Done. The app will work normally from here.
