About the Software

The Flexible Pavement Carbon Footprint Estimator is a desktop application that calculates layer-wise and total carbon emissions using AI/ML models. It is designed for civil engineers, academicians, and pavement researchers to simplify carbon estimation for sustainable pavement design.

Features

  • AI/ML-based carbon emission prediction
  • Fully offline usage – no internet required
  • Automatic backend startup
  • Layer-wise & total emission breakdown
  • Professional PDF report generation
  • User-friendly interface
